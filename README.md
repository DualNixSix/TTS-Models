# TTS-Models

#### Models can be imported into Table Top Simulator:
```
1. From the main menu create a new Single-Player game, select Modeling > Cloud Manager and upload desired custom assets from local computer to cloud.

2. Navigate to the top of the screen and select Objects > Components > Custom > Custom Model.

3. Drag the Custom Model into the game and a settings menu will open.

4. Select the desired object .obj file in Modeling > Cloud Manager and it will copy a link to the clipboard for placing into Model/Mesh.

5. Select the desired color .png file in Modeling > Cloud Manager and it will copy a link to the clipboard for placing into Diffuse/Image.

6. Once desired object has been set, modify other settings under Model and Material as needed, then click import.

7. Once imported right click the newly created object and click save object. Object will be saved to root folder and found in Objects > Saved Objects. 

8. Play and Enjoy!
```
